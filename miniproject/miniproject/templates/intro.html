Speculation of Flight Ticket Price using machine learning

What is FairFare

FairFare is a web application which simulates various models for computing expected future prices of Airlines and provide user with the information about the best time to buy the ticket. User can make use of this information for their future travel plans. This application provide user a pridicted price for the traivel information given by them.

Why we need FairFare

As domestic air travel is getting more and more popular these days in India with various air ticket booking channels coming up online, travellers are trying to understand how these airline companies make decisions regarding ticket prices over time.Airline corporations are using complex strategies and methods to assign airfare prices in a dynamic. So, there is a need for predicting the price of the air tickets, which helps the customer to take the decision.

How it works

Machine Learning algorithms are applied on the dataset to predict the dynamic fare of flights. This gives the predicted values of flight fare to get a flight ticket at minimum cost. To develop the model for the flight price prediction, many conventional machine learning algorithms are considered and evaluated. The different models which can may be used are Linear regression, Decision tree, Random Forest Algorithm, K-Nearest neighbors.
